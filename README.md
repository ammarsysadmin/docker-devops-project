# 🚀 Production-Ready Docker DevOps Project

## 📌 Overview
This project is a production-style DevOps system using Docker, Docker Compose, CI/CD (Jenkins), Security scanning (Trivy), and Monitoring (Prometheus + Grafana).

It demonstrates real-world DevOps workflow from development → deployment → monitoring.

---

## 🏗️ Architecture

User → Nginx → Frontend → Backend → Database  
                       ↓  
        Monitoring (Prometheus + Grafana)  
        CI/CD (Jenkins)  
        Security (Trivy)

---

## ⚙️ Technologies Used

- Docker (Alpine Images)
- Docker Compose
- Node.js (Backend API)
- Nginx (Reverse Proxy)
- MongoDB (Database)
- Jenkins (CI/CD Pipeline)
- Trivy (Security Scanner)
- Prometheus (Monitoring)
- Grafana (Dashboard)
